# Queue & Stack Problems

The goal of this category is to strengthen problem-solving fundamentals and build clean, readable, beginner-friendly solutions.

1. Open `Data-Structures-and-Algorithms-Practice.sln`
2. Select any project inside `01-Queue-Stack`
3. Set it as the startup project
4. Run it from Visual Studio
