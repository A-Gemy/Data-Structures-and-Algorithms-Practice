# Dictionary & HashSet Problems

This folder contains 20 C# console problems focused on practicing Dictionary and HashSet data structures.

The goal of this category is to strengthen problem-solving skills and build clean, readable, beginner-friendly solutions using fast lookup and counting techniques.

## Problems

- DH-01 Store and retrieve student grades
- DH-02 Store book information
- DH-03 Translate words between languages
- DH-04 Count word frequencies in text
- DH-05 Phonebook application
- DH-06 Track unique website visitors
- DH-07 Check duplicate entries in real time
- DH-08 Dynamic skill matching
- DH-09 Find character frequencies
- DH-10 Find longest consecutive sequence
- DH-11 Find majority element
- DH-12 Find duplicate elements
- DH-13 Find all unique elements
- DH-14 Find words typed using one keyboard row
- DH-15 Find missing number in array
- DH-16 Find common characters in strings
- DH-17 Find elements not in second array
- DH-18 Find disappeared numbers in array
- DH-19 Check if two arrays are disjoint
- DH-20 Check if a sentence is pangram

## How to Run

1. Open `Data-Structures-and-Algorithms-Practice.sln`
2. Select any project inside `02-Dictionary-HashSet`
3. Set it as the startup project
4. Run it from Visual Studio
