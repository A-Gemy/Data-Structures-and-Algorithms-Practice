# SortedList & SortedSet Problems

This folder contains 11 C# console problems focused on practicing SortedList and SortedSet data structures.

The goal of this category is to understand sorted collections, range operations, automatic ordering, and key/value storage using clean and readable solutions.

## Problems

- SL-01 Find elements in range
- SL-02 Find smallest and largest element
- SL-03 Find union of two SortedSets
- SL-04 Remove elements greater than value
- SL-05 Find elements less than value
- SL-06 Count elements greater than value
- SL-07 Remove elements within range
- SL-08 Find elements not in range
- SL-09 Count elements less than or equal to value
- SL-10 Check if set contains multiple ranges
- SL-11 Real-time leaderboard

## How to Run

1. Open `Data-Structures-and-Algorithms-Practice.sln`
2. Select any project inside `03-SortedList`
3. Set it as the startup project
4. Run it from Visual Studio
